AI Toolkit Resources

A guide for educators on how to build simple web applications using AI tools.

Getting Started

Tools: It recommends using an AI-powered code editor called Cursor. It also provides setup instructions for standard code editors on both macOS (TextEdit, Visual Studio Code) and Windows (Notepad, Visual Studio Code).

Process: The basic workflow involves creating a new HTML file (e.g., myFirstApp.html) in one of these editors.

Prompting and App Creation

Prompting Guide: The document advises users to be specific when writing prompts for an AI model. It suggests a structure for prompts that outlines what the app does, what it should look like, and how users interact with it.

Example Prompt: A detailed example is provided for creating a "vocabulary quiz application," specifying features like data entry, local storage, and an edit mode.

Workflow: After submitting a prompt to an AI model, the user should copy the generated code, paste it into their editor, save it as an HTML file, and open it in a web browser.

Application Examples

Mild Examples (Simple, single-file apps)

  • Pomodoro Timer: A focus timer that warns the user if their mouse leaves a designated area.
  • Word Reader: Displays pasted text one word at a time to aid focus.
  • Task List Maker: An organization tool to sort tasks into categories like "Now," "Today," and "This week."

Medium Examples (More complex, using local storage)

  • IEP Tracker: A tool for special education teachers to track IEP deadlines, reminders, and tasks for multiple students.

Advanced Examples (Using APIs and complex logic)

  • AI Text Writer: A distraction-free word editor that uses the Google Gemini API to provide AI-generated encouragement.

Advanced Concepts and Tools

"Vibe Coding": This term describes using an AI agent (like lovable.dev) to build an application based on verbal or written feedback, without the user needing to write or paste code themselves.

AI Chat Options: The document compares several platforms for generating code:

  • Google AI Studio: Recommended for free access to powerful models like Gemini 1.5 Pro.
  • Gemini Chat: A user-friendly interface for Google's models.
  • ChatGPT: The well-known tool from OpenAI, with free and premium (Plus) versions.
  • OpenAI Playground: A pay-as-you-go option offering more control over models, settings, and system messages.

AI Terminology

Models: Different AI versions.

Model Settings: e.g., "temperature" for creativity.

System Messages: Instructions that define the AI's role.

API Keys

To add AI features to a custom app, the document explains that an API key is required from a provider like Google (for Gemini) and provides a link to obtain one.
